для просмотра плагинов, ничего не делайте, меня заебало между ветками переключаться

# список плагинов
1. **fix_endpoint** - изменяет конечную точку аватарки в профиле, путем редактирования "капли"
2. **fuck_gifts** - отключает кнопки дарения, диалоги отправки подарков, показ подарков в профиле, диалоги просмотра подарков и т.п. (пока не скрывает подарки в чате)
3. **shortcuts** - ярлычки на вкл/откл плагина, настройки плагина, и позволяет переключать тумблеры настроек в плагинах (не везде, и именно тумблеры)
4. **говорим ,kjznm правильно** - переписывает маты, путем написания на англ. раскладке, *часть сделки с __хоприк плагинс__*
5. **Hide Slash** - скрывает кнопку "слеша" в групповых чатах (было сделано по просьбе)

Плагины публуются в [инструментах для вящеВестника](https://t.me/exteraPluginsSup), KPM a.k.a. Kangel Plugins Manager ([Скачать плагин](https://t.me/exteraPluginsSup/407), [Форум плагина](https://t.me/KangelPluginsManager)), [exteraStore](https://exterastore.app/developers/1302340931), и самыми первыми публикуются в [feature plugins](https://t.me/feature_plugins) (ну и в принципе они там все и валяются)
